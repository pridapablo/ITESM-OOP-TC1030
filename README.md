# ITESM-OOP-TC1030

## Course Information
**Institution:** Tecnológico de Monterrey  
**Course:** TC1030 - Object-Oriented Programming  
**Website:** [TC1030 Course Overview](https://samp.itesm.mx/Materias/VistaPreliminarMateria?clave=TC1030)

The ITESM-OOP-TC1030 repository is dedicated to the coursework and programming assignments of the TC1030 Object-Oriented Programming class. It includes a variety of projects and exercises focused on advanced object-oriented programming concepts, inheritance, polymorphism, operator overloading, and exception handling in C++.

## Course Description
The TC1030 course delves into the object-oriented programming paradigm, exploring advanced features of C++ to solve complex computational problems. The course emphasizes the assessment of problem components, decision-making in uncertain conditions, and the application of research and computational methodologies.

## Learning Objectives
Upon completing the course, students are expected to:

- Evaluate problem components using computational principles and processes.
- Make informed decisions in problem-solving within uncertain and complex situations.
- Implement object-oriented designs for software solutions.

## Repository Contents
This repository includes:

- Object-Oriented Design and Programming Concepts
- Inheritance and Polymorphism
- Access Modifiers, Method Overloading, and Overriding
- Abstract Classes and Interfaces
- Operator Overloading in C++
- Exception Handling Mechanisms

## Suggested Bibliography
- Deitel, Paul J. _C++: How to Program_, 9th ed., Prentice Hall, 2014.
- Stroustrup, Bjarne. _The C++ Programming Language_, 4th ed.
- Sahay, Sourav. _Object-oriented programming with C++_, Oxford University Press, 2006.
